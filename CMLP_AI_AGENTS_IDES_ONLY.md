## QUICK START GUIDE

### Konfiguracja IDE dla CMLP:

**Opcja A: Cursor (Rekomendowane)**
1. Pobierz Cursor: https://cursor.sh
2. Otwórz folder `CMLP HardbanRecordsLab`
3. Zainstaluj extensions: ESLint, Prettier, Tailwind CSS, Drizzle
4. Skonfiguruj AI: Settings → AI → wybierz model (Claude/GPT-4/Gemini)
5. Ready to code!

**Opcja B: VS Code + Copilot + Continue**
1. Zainstaluj VS Code: https://code.visualstudio.com
2. Extensions: GitHub Copilot, Continue.dev, ESLint, Prettier
3. Skonfiguruj `.env` z kluczami API
4. Ready to code!

**Opcja C: GitHub Codespaces (Cloud)**
1. Otwórz repo w GitHub
2. Kliknij "Code" → "Codespaces" → "Create codespace"
3. Gotowe — wszystko w chmurze, zero lokalnego setup.
